# /minutes - 議事録テンプレート生成コマンド

アジェンダまたはMTG情報から議事録のテンプレートを生成します。

## 実行手順

### Step 1: 入力情報の確認

ユーザーに以下を確認します：

1. **アジェンダがある場合**: アジェンダをペーストしてもらう
2. **アジェンダがない場合**: 以下を確認
   - MTGの種類
   - MTG名/タイトル
   - 参加者（任意）
   - 主な議題（任意）

### Step 2: 出力形式を確認

「出力形式はどれにしますか？」と質問します：

1. **Markdown**（デフォルト）
2. **Slack**
3. **Notion**
4. **Plain**
5. **HTML**

### Step 3: 議事録テンプレートを生成

---

## 議事録テンプレート構成

### 基本構成

1. **ヘッダー情報**
   - MTG名
   - 日時
   - 参加者
   - 記録者

2. **議題ごとの記録欄**
   - 議題名
   - 議論内容
   - 決定事項
   - 保留事項

3. **アクションアイテム**
   - タスク内容
   - 担当者
   - 期限

4. **次回MTGについて**
   - 日時
   - 議題候補

## MTGタイプ別の追加セクション

### 1on1の場合
```
## フォローアップ事項
- 前回からの進捗:
- 継続して確認すること:

## メンバーの状態
- モチベーション:
- 困りごと:
- サポートが必要なこと:
```

### 振り返り（レトロ）の場合
```
## KPT

### Keep（続けること）
-

### Problem（課題）
-

### Try（次に試すこと）
-
```

### 意思決定会議の場合
```
## 決定事項サマリー

| 項目 | 決定内容 | 理由 |
|------|----------|------|
|  |  |  |

## 却下された選択肢
- 選択肢A: （却下理由）
```

---

## 注意事項

- 議事録は可能な限りMTG中にリアルタイムで記録することを推奨
- 決定事項とアクションアイテムは必ず明確に記載
- 参加者全員がアクセスできる場所に共有してください
