# meeting-agenda

対話形式でMTGアジェンダを作成するプラグインです。全てのMTGタイプに対応し、Slack/Notion/Markdown等の出力形式を選択できます。

## 機能

- 対話形式でアジェンダを作成
- 8種類のMTGタイプに対応
- 5種類の出力フォーマット
- 議事録テンプレート生成
- ファシリテーション知識ベース

## インストール

```bash
/plugin install meeting-agenda@my-claude-plugins
```

## コマンド

### `/agenda` - アジェンダ作成

対話形式でMTGアジェンダを作成します。

```bash
/agenda
```

**対話の流れ:**
1. MTGの種類を選択
2. 基本情報を入力（時間、人数、目的）
3. 出力形式を選択
4. アジェンダが生成される

### `/minutes` - 議事録テンプレート生成

アジェンダから議事録のテンプレートを生成します。

```bash
/minutes
```

## 対応MTGタイプ

| タイプ | 説明 |
|--------|------|
| 定例ミーティング | 週次、デイリー、月次などの定期MTG |
| 1on1 | マネージャーとメンバーの1対1面談 |
| ブレインストーミング | アイデア出し、企画会議 |
| 意思決定会議 | 承認、レビュー、決定が必要な会議 |
| 振り返り（レトロ） | KPT、Start/Stop/Continue |
| キックオフ | プロジェクト開始、新規施策立ち上げ |
| 報告会 | 進捗報告、成果共有 |
| ワークショップ | ハンズオン、研修 |

## 出力フォーマット

| フォーマット | 用途 |
|-------------|------|
| Markdown | GitHub、ドキュメント |
| Slack | Slack投稿（mrkdwn形式） |
| Notion | Notionページ |
| Plain | メール、チャット全般 |
| HTML | リッチメール（Outlook等） |

## 出力例

### Slack形式

```
:calendar: *スプリントレビュー*
:clock1: 60分 | :busts_in_silhouette: 5名

*アジェンダ*
1️⃣ 開会・ゴール確認（5分）
2️⃣ 成果デモ（25分）
3️⃣ 課題共有（15分）
4️⃣ 次スプリントへの申し送り（10分）
5️⃣ クロージング（5分）

*事前準備*
• デモ環境の動作確認
• 各自の成果物まとめ
```

### Markdown形式

```markdown
## スプリントレビュー

📅 **日時**: （要記入）
⏱ **所要時間**: 60分
👥 **参加者**: 5名

### アジェンダ

| # | 議題 | 時間 | 担当 |
|---|------|------|------|
| 1 | 開会・ゴール確認 | 5分 | ファシリテーター |
| 2 | 成果デモ | 25分 | 各担当者 |
| 3 | 課題共有 | 15分 | 全員 |
| 4 | 次スプリントへの申し送り | 10分 | PO |
| 5 | クロージング | 5分 | ファシリテーター |

### 事前準備
- [ ] デモ環境の動作確認
- [ ] 各自の成果物まとめ
```
