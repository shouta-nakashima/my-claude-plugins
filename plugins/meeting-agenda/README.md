# Meeting Agenda

対話形式でMTGアジェンダを作成するClaude Codeプラグインです。

## 概要

ミーティングの準備を効率化します：
- 自由に説明するか、テンプレートから選ぶかを選択可能
- 8種類のMTGタイプに対応したアジェンダを自動生成
- Slack、Notion、Markdownなど5種類の出力形式をサポート
- 議事録テンプレートも生成可能

## 機能

### コマンド

| コマンド | 説明 |
|----------|------|
| `/agenda` | 対話形式でアジェンダを作成 |
| `/minutes` | 議事録テンプレートを生成 |

### スキル

MTGファシリテーションに関する知識ベースを提供。効果的なMTGの原則、タイプ別のベストプラクティス、タイムマネジメントのコツなどを参照できます。

## インストール

```bash
# マーケットプレイスを追加（初回のみ）
/plugin marketplace add https://github.com/shouta-nakashima/my-claude-plugins

# プラグインをインストール
/plugin install meeting-agenda@my-claude-plugins
```

## 使い方

### アジェンダ作成

```bash
/agenda
```

1. 作成方法を選択
   - **自由に説明する**: MTGの内容を自由に入力
   - **テンプレートから選ぶ**: 8種類のテンプレートから選択
2. 不足情報を確認（所要時間、参加人数）
3. 出力形式を選択
4. アジェンダが生成される

### 議事録テンプレート生成

```bash
/minutes
```

1. アジェンダをペースト、またはMTG情報を入力
2. 出力形式を選択
3. 議事録テンプレートが生成される

## 対応MTGタイプ

| タイプ | 説明 |
|--------|------|
| 定例ミーティング | 週次、デイリー、月次などの定期MTG |
| 1on1 | マネージャーとメンバーの1対1面談 |
| ブレインストーミング | アイデア出し、企画会議 |
| 意思決定会議 | 承認、レビュー、決定が必要な会議 |
| 振り返り（レトロ） | KPT、Start/Stop/Continue |
| キックオフ | プロジェクト開始、新規施策立ち上げ |
| 報告会 | 進捗報告、成果共有 |
| ワークショップ | ハンズオン、研修 |

## 出力フォーマット

| フォーマット | 用途 |
|-------------|------|
| Markdown | GitHub、ドキュメント向け |
| Slack | Slack投稿用（mrkdwn形式） |
| Notion | Notionページ向け |
| Plain | メール、チャット全般 |
| HTML | リッチメール（Outlook等） |

## 例

### アジェンダ作成の例

```
User: /agenda
Claude: アジェンダの作成方法を選んでください
        1. 自由に説明する
        2. テンプレートから選ぶ
User: 1
Claude: どんなMTGですか？自由に教えてください
User: 来週のスプリントレビュー、1時間で開発チーム5人参加
Claude: 出力形式はどれにしますか？（デフォルト: Markdown）
User: Slack

[Claudeがアジェンダを生成...]
```

### Slack形式の出力例

```
:calendar: *スプリントレビュー*
:clock1: 60分 | :busts_in_silhouette: 5名

*アジェンダ*
1️⃣ 開会・ゴール確認（5分）
2️⃣ 成果デモ（25分）
3️⃣ 課題共有（15分）
4️⃣ 次スプリントへの申し送り（10分）
5️⃣ クロージング（5分）

*事前準備*
• デモ環境の動作確認
• 各自の成果物まとめ
```

### Markdown形式の出力例

```markdown
## スプリントレビュー

📅 **日時**: （要記入）
⏱ **所要時間**: 60分
👥 **参加者**: 5名

### アジェンダ

| # | 議題 | 時間 | 担当 |
|---|------|------|------|
| 1 | 開会・ゴール確認 | 5分 | ファシリテーター |
| 2 | 成果デモ | 25分 | 各担当者 |
| 3 | 課題共有 | 15分 | 全員 |
| 4 | 次スプリントへの申し送り | 10分 | PO |
| 5 | クロージング | 5分 | ファシリテーター |

### 事前準備
- [ ] デモ環境の動作確認
- [ ] 各自の成果物まとめ
```

### テンプレートから選ぶ例

```
User: /agenda
Claude: アジェンダの作成方法を選んでください
        1. 自由に説明する
        2. テンプレートから選ぶ
User: 2
Claude: テンプレートを選んでください
        1. 定例ミーティング
        2. 1on1
        3. ブレインストーミング
        ...
User: 5
Claude: 振り返り（レトロ）ですね。所要時間は？
User: 1時間
Claude: 参加人数は？
User: 8人
Claude: 出力形式はどれにしますか？
User: Notion

[Claudeが振り返り用アジェンダを生成...]
```

## 利点

- **時間短縮**: アジェンダ作成にかかる時間を大幅に削減
- **網羅性**: MTGタイプに応じた必要な議題を漏れなくカバー
- **柔軟性**: 自由記述とテンプレート選択の両方に対応
- **出力形式**: 利用シーンに合わせた5種類のフォーマット
- **知識ベース**: ファシリテーションのベストプラクティスを参照可能

## 注意事項

- 時間配分は目安です。MTGの性質に応じて調整してください
- 1時間を超えるMTGの場合は、休憩時間が自動で提案されます
- 出力形式を指定しない場合、Markdownがデフォルトで使用されます
