# /coaching コマンド

対話型のコーチングセッションを開始します。

## 使い方

```
/coaching
```

## 実行内容

このコマンドを実行すると、AIコーチとの対話セッションが始まります。

### セッションの流れ

1. **テーマのヒアリング**
   - 今日話したいテーマを聞きます
   - 仕事、キャリア、人間関係、目標、モヤモヤなど何でもOK

2. **対話セッション**
   - テーマに合わせて最適な対話スタイルで進めます
   - 質問を通じて、あなた自身の考えを整理します

3. **まとめ**
   - 対話で出てきた気づきや決めたことを整理します
   - 次のアクションがあれば確認します

## 対応できるテーマ例

- 目標を達成したい
- キャリアについて考えたい
- 決断できないことがある
- 考えを整理したい
- モヤモヤを言語化したい
- 新しいことに挑戦したい

## 注意事項

- コーチングは「答えを教える」ではなく「答えを引き出す」アプローチです
- 専門的な医療・法律・財務のアドバイスはできません
- 深刻な悩みの場合は、専門家への相談をおすすめすることがあります
