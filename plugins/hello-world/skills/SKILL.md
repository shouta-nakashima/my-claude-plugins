# プロジェクト情報表示スキル

このスキルは、ユーザーが現在のプロジェクトに関する情報を求めたときに起動します。

## 使用するタイミング

以下のような質問があったときに使用してください：
- "このプロジェクトについて教えて"
- "リポジトリの情報は？"
- "何のプロジェクト？"
- "プロジェクト概要を表示"

## 実行内容

1. README.mdファイルを読み込んで概要を取得
2. package.jsonやその他の設定ファイルから技術スタックを特定
3. ディレクトリ構造を分析
4. 以下の情報を整理して表示：
   - プロジェクト名
   - 説明
   - 使用技術
   - 主要な機能
   - ディレクトリ構成

## 出力形式

わかりやすく構造化された形式で情報を提示してください。マークダウン形式を使用し、見出しや箇条書きを活用してください。

## 例

```
# プロジェクト: My App

## 概要
ユーザー管理機能を持つWebアプリケーション

## 技術スタック
- Frontend: React, TypeScript
- Backend: Node.js, Express
- Database: PostgreSQL

## 主要機能
- ユーザー認証
- プロフィール管理
- データダッシュボード
```
